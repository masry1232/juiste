                Managing Customer Returns

                USPS Return Services Tips & Tools Help Reduce Costs

                Make it easy for customers to return packages, whether they value speed via Priority Mail Express® Return or Priority Mail Return® services, or more affordable pricing with USPS Ground Advantage® Return. Build customer loyalty, increase sales, and control costs with flexible USPS® options.

                Ways to Provide Return Labels

                • Request USPS Label Delivery service through Click-N-Ship® or USPS APIs. We'll print and deliver the physical labels to you for a $1.55 fee (per label).
                • Include one in the original shipment box.
                • Use the cloud platform.
                • Integrate the returns tool on your website.
                • Let customers print their own.

                Compare Return Options

                USPS offers several commercial pricing methods for managing customer returns.
                Returns Services Prices

                Notice on Noncompliance Fees: Mailers using all return services must comply with rules on the accuracy of the Intelligent Mail® package barcode (IMpb®) and include a matching Intelligent Mail Matrix Barcode (IMmb) on their postage labels. Failure to meet the Package Quality compliance threshold will result in the assessment of a $0.25 per-label noncompliance fee. For more information, please refer to DMM section 204.2.1.8.

                Priority Mail Express Return

                Priority Mail Express Return provides 1–3 day delivery of packages up to 70 lbs. It's a fast and affordable premium solution for shippers who want to provide prepaid return labels to their customers. This service includes $100 insurance, and for an additional fee, you can get insurance up to $5,000 against loss and damage.

                See Pricing Files

                Priority Mail Return

                Priority Mail Return offers a combination of speed and value that is ideal for time-sensitive or high value items. It's a fast and affordable premium solution for high-volume shippers who want to provide prepaid return labels to their customers. This service includes $100 insurance, and for an additional fee, you can get insurance up to $5,000 against loss and damage.

                See Pricing Files

                USPS Ground Advantage Return

                USPS Ground Advantage Return offers your customers an affordable and reliable way to send their items back to you within 2-5 days1. It's the primary option for ground shipping (including HAZMAT that cant travel by air). This service includes $100 insurance, and for an additional fee, you can get insurance up to $5,000 against loss and damage.

                See Pricing Files

                Need Help?

                For assistance with the USPS-hosted label generation tool or if you would like more information on returns, contact the USPS Mailing & Shipping Solutions Center (MSSC) Help Desk at 1-877-672-0007.

                Need to return something you purchased?

                Easily create and print a return label with Click-N-Ship. To get started, sign in to your USPS.com account or sign up for free today.
